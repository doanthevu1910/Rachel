# macOS Monterey 12.6.3 / Clover 5150

# Works
- Sleep / Wake
- Audio
- Ethernet
- All USB ports (2.0, 3.0 & Type-C)
- All Apple services (FaceTime, Find My, iMessage, iCloud)

# Doesn't work
- AirDrop, Handoff, Continuity (native Wifi/Bluetooth card needed)
